# Fonts Directory

Please download the following fonts and place them in this directory:

## DM Serif Display
- **Source**: [Google Fonts](https://fonts.google.com/specimen/DM+Serif+Display)
- **Weights**: 400 (regular), 400 italic
- **Format**: WOFF2
- **File name**: `dm-serif-display.woff2`

## DM Sans
- **Source**: [Google Fonts](https://fonts.google.com/specimen/DM+Sans)
- **Weights**: 300, 400, 500, 600, 700
- **Format**: WOFF2 Variable Font
- **File name**: `dm-sans.woff2`

## JetBrains Mono
- **Source**: [Google Fonts](https://fonts.google.com/specimen/JetBrains+Mono)
- **Weights**: 400, 500, 600
- **Format**: WOFF2 Variable Font
- **File name**: `jetbrains-mono.woff2`

## How to Download

### Option 1: Google Fonts
1. Visit each font's Google Fonts page
2. Click "Download family"
3. Use a tool like [google-webfonts-helper](https://gwfh.mranftl.com/fonts) to get WOFF2 formats

### Option 2: Manual Download
```bash
# Or use a font downloader tool
# Example: fontsource packages (npm)
```

## Expected Files

After downloading, you should have:
```
public/fonts/
├── README.md (this file)
├── dm-serif-display.woff2
├── dm-sans.woff2
└── jetbrains-mono.woff2
```
